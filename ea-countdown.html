Sprint Countdown Timer

A small web app displayed on large screen TVs counting down the number of days remaining in the sprint. This would remind the busy engineering team of where they were in the work cycle.

 

One time setup

Only two values are needed to begin: the sprint duration AND any previous start date. So, if you start on an odd week, or if you start on a Tuesday, just enter that.

 
$( document ).ready(function() { var cadence = 10; // 10 day sprint duration var anyPreviousSprintStartDate = new Date("06/02/2014"); ...
 

And no maintenance..

The algorithm will automatically calculate the days remaining in the sprint based on the two values entered. The page will update on its own without further intervention.

 
... var now = new Date(); var next, workingDays; // Calculate sprint days remaining based on cadence and any previous sprint start date workingDays = workingDaysBetweenDates(anyPreviousSprintStartDate, now); workingDays = cadence - ((workingDays-1) % cadence); // 1 offset: don't count current day ...
